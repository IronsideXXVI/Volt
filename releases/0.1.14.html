Volt 0.1.14

Rebuild SettingsView entirely on the token system used by the popover so the
two surfaces read as siblings:

  • Replace every inline font/color with tokens. Add two field-type tokens,
    voltControlLabel (12 semibold) and voltChipText (11 semibold), reusing the
    existing 12/11 size tiers rather than introducing new sizes.
  • Collapse to one accent: primary everywhere, orange for errors, green only in
    the Updates pane. Drop per-provider pane tints and sidebar status dots.
  • Flatten the sidebar and drive provider rows/marks with the template logos
    (new VoltLogoGlyph), matching the popover switcher. Convey connection state
    through the title color instead of a redundant chip.
  • Route SectionHeader through voltSectionHeader/voltCaption.

Credential import/save-and-test/disconnect/dirty-state logic is unchanged.
Document the Settings surface and the two tokens in the design system.
