Volt 0.1.18

Volt is an agent app (LSUIElement), so SettingsLink opened the Settings window
behind whatever app was frontmost and left the menu-bar popover up. Replace the
three SettingsLink buttons with a shared handler that calls
NSApp.activate(ignoringOtherApps:) before openSettings(): the app comes to the
foreground, and as focus moves to the Settings window the popover dismisses.
